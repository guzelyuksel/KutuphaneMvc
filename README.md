# KutuphaneMvc

ASP.NET MVC Core, Entity Framework, Bogus kullanarak eğitim için yapılmış ufak bir otomasyon. 
Eksikleri çok belki zamanla, öğrendikçe, tamamlanabilir.

Çalıştırmak için Sql Local Db kuruluysa Update-Database komutu yeterli olacaktır. 
Yüklü değilse connection string değiştirilerek kullanılabilir.
